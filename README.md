# Double-Jump-mc-MOD
This mod adds strategic mobility while preventing abuse through the stamina system and cooldowns. Players must manage their stamina carefully during exploration and combat!
# Double Jump Mod for Minecraft 1.8.8

## Features
- ✅ Double jumps while in mid-air
- ✅ Stamina system with visual HUD
- ✅ Stamina regeneration
- ✅ Manual recharge (Press R)
- ✅ Configurable jump strength
- ✅ No server-side required (client-side only)

## Installation
1. Install Minecraft Forge for 1.8.8
2. Download the mod JAR
3. Place in `.minecraft/mods/` folder
4. Launch Minecraft with Forge profile

## Controls
- **Double Jump:** Press Spacebar while in air
- **Recharge Stamina:** R (while on ground)
- **Stamina Cost:** 25 per double jump
- **Max Stamina:** 100

## Compatibility
- Minecraft: 1.8.8
- Forge: 11.15.1.2318
- Client-side only
- No conflicts with other mods
